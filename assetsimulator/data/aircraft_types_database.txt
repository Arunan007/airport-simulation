# All data may be entered in upper or lower case.
# Model type is the string to use in an aircraft list file or in the Aircraft Creator.
# Default weight class may only be large, heavy, or small.
# Climb and descent rates must be positive and are in feet per minute. Descent rate value 
# is assumed to be the maximum descent rate.
# Number of engines must be greater than 0.
# Engine type may only be 'j' for jet 't' for jet/turboprop and 'p' for piston.
# Minimum airspeed, in knots, must be greater than 0 and less than maximum airspeed.
# Maximum airspeed, in knots, must be greater than 0 and greater than minimum airspeed.
# Maximum altitude, in feet, must be greater than 0.
# Default climb speed, in knots, must be between the minimum and maximum airspeed.
# Default descent speed, in knots, must be between the minimum and maximum airspeed.
# Max TRACON speed, in knots,  maximum speed that an aircraft should be at within 
# 40 miles from touchdown
# Max apprch speed, in knots, maximum speed an aircraft will be automatically 
# assigned when within "Strait in dist" distance for straight-in situtions and 
# "Base leg dist" distance when in a base-leg sitution.
# Strait in dist, distance in nautical miles, distance from touchdown to reduce 
# and aircraft to "Max apprch speed" when the aircraft is in a straight-in sitution.
# Base leg dist, distance in nautical miles, distance from touchdown to reduce 
# and aircraft to "Max apprch speed" when the aircraft is in a base-leg sitution.
# VFR decel distance, distance in nautical miles, minimum deceleration distance
# for a VFR approach 
# Pseudo model is the model pseudocontrol will use to fly the aircraft.  The name of
